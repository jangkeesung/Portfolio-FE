주제 : 동물과 사람들이 어울리는 공간을 만들어주는 펫 관련 플랫폼 개발

개발 환경 :
Platform Windows 10 Intel(R) i5-8250 CPU, 8GB, x64
Language(version) JAVA(JDK 11.0.2), SQL, HTML5, CSS, JavaScript, JSP
IDE Eclipse 4.19.0 (2021-03), VSCode, SQL Developer
Used Skills JDBC, jQuery, Bootstrap, Servlet, Ajax
WAS Apache tomcat-8.5.87
Database Oracle, SQL-Server, PL-SQL

개발 기간 : 2023.05.24 ~ 2023.06.12

목적 : 반려동물 관련 서비스에 대한 수요가 높아짐에 따라 이를 충족시키기 위한 다양한 기능을 통합한 플랫폼을 개발

개요 :

  • 기존 범용 시스템의 한계 개선
    • - 여러 사이트에 갈 필요 없이 한 사이트에서 다양한 서비스를 이용할 수 있는 기능 제공
  • 현행 플랫폼의 신뢰성 부족 개선
    • - 데이터의 정확성과 최신성을 보장하는 기능 제공
  • 1인 가구 증가로 인한 반려동물 케어 시간 부족 개선
    • - 비용 부담을 줄이고 신뢰할 수 있는 펫시터를 찾을 수 있는 기능 제공
  • 반려인들의 소통의 어려움 개선
    • - 사용자들이 활발한 소통을 할 수 있게 하는 기능 제공

구현 기능 :

  • 로그인, 회원가입, 소셜로그인(카카오)
  • 병원/약국 관련 정보 조회
    • - 카카오맵 API를 활용한 동물병원/동물약국 선택 후 위치 및 정보 조회 기능
    - 특정 병원 또는 약국 선택 후 리뷰 작성 및 조회 기능
    - 지적편집도·지형정보·교통정보·자전거도로 정보 조회 기능
  • 유기동물 입양 공고 확인
    • - 유기동물 입양 공고 조회
    - 서울특별시 공공데이터(서울동물복지지원센터 입양 대기동물 현황)를 활용하여 입양 대기 중인 유기동물 공고 조회 기능
  • 실종 동물 찾기

웹을 배우고나서 첫 프로젝트로 프론트엔드 페이지 제작과 서버로의 데이터 전송 및 호출을 구현해보았다.

게시판 CRUD 기능과 검색 및 페이징 기능을 기본적으로 구현하였으며, 회원가입 및 로그인 또한 구현해볼 수 있었다.

게시판의 글 내용은 CKEditor Library를 이용하여 약간의 글 꾸미기 및 이미지 삽입 처리까지 하였다.

'회원가입 및 정보 수정' 시 주소 검색 API를, '병원/약국 찾기 및 산책친구'에서 카카오맵 API를, '당신을 기다려요'에서 유기동물 입양 정보 API를 적극 활용하여 외부 API 기능 사용법을 잘 익힐 수 있었다.

JSP 페이지에서 필요한 데이터를 Servlet과의 연동을 통해 호출 또는 전송하여 JDBC로 데이터베이스에 접근, 데이터를 조작하고 select한 데이터를 다시 JSP 페이지에서 jstl과 el로 출력하는 훈련을 할 수 있었다.

자바, 오라클, 웹까지 배운 모든 것을 활용하여 실제 사이트처럼 구현해보니 우리가 이용하는 사이트들이 얼마나 많은 정보 처리가 이루어지고 있었는지 알 수 있었다.
